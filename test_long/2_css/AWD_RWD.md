## CSS 속성 및 분류
* css작성시 작성 순서 고려
  * CSS 에 속성을 선언하는데에는 암묵적·묵시적인 선언 순서가 있습니다.
  * 다시 말해, 속성과 값을 선언하는데 정확한 순서는 없지만 일반적인 컨벤션이 있습니다.
  * 컨벤션이라 함은 관례, 관습의 의미하는데 프로그래밍적으로는 암묵적·묵시적인 개발자 서로 간의 약속과 같은 것을 말합니다.
